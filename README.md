# cadastro_clientes
Esse sistema é de cadastro de clientes, realiza o CRUD completo.

- Cadastrar cliente: Insere um cliente na tabela. 
  Entrada: Cliente. 
  Retorno: Cliente que foi inserido na tabela.
  
- Update cliente: Altera os dados do cliente na tabela.
  Entrada: Cliente a ser alterado.
  Retorno: Cliente que foi alterado.
  
- Delete cliente: Deleta um cliente da tabela.
  Entrada: ID do cliente a ser deletado.
  Retorno: Mensagem de sucesso de deleção.
  
- Retorna clientes: Busca todos os clientes da tabela.
  Entrada: Nenhuma.
  Retorno: Clientes da tabela.
